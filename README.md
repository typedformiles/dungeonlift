# DungeonLift

A pixel-art dungeon crawler built as a GTM tool for [Neuralift](https://neuralift.ai) — the AI-powered customer segmentation platform.

**[Play now](https://typedformiles.github.io/dungeonlift/)**

## The Game

You're a data professional descending through the Data Dungeon, fighting the enemies of insight and discovering hidden customer segments. Find Neuralift to unlock everything.

### Choose Your Hero
- **Alex Chen** — Data Analyst
- **Jordan Blake** — Marketing Executive
- **Dr. Maya Patel** — Data Scientist
- **Sarah Owens** — Chief Experience Officer

### Enemies
| Enemy | What it represents |
|-------|--------------------|
| Noise | Random noise obscuring the signal |
| Bias | Human bias distorting the truth |
| Data Silo | Siloed data blocking the full picture |
| Rules Engine | Rigid logic missing what matters |
| Spreadsheet | Death by spreadsheet |
| SQL QUERY (Boss) | The limits of manual querying |

### Mechanics
- **6 dungeon levels** with procedurally generated rooms and corridors
- **Turn-based RPG combat** against data/analytics themed enemies
- **Quiz-gated segments** — answer data & marketing questions to unlock discoveries
- **Inventory system** with ML-themed items: GPU Shield, Denoiser, Neural Cloak, Batch Normalizer, Hyperparameterizer, and more
- **Neuralift power-up** — the ultimate discovery that transforms your character and clears the level
- **Share your score** — Wordle-style clipboard scorecard and LinkedIn share

### Controls
| Key | Action |
|-----|--------|
| Arrow keys / WASD | Move (explore) and navigate (battle) |
| Enter / Space | Confirm action |
| I or E | Open inventory |
| 1-4 | Answer quiz questions |

## Tech

Pure HTML5 Canvas + vanilla JS. No dependencies, no build step. Runs entirely client-side.

## About Neuralift

Neuralift is a customer and business context engine that sits on top of the data warehouse. It uses deep learning to discover customer segments that rules-based tools miss — no SQL, no bias, no guesswork.
